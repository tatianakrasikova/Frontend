Lesson 04

Flexbox - это гибкая технология для позиционирования элементов в CSS

  • Чтобы "включить" flex отображение в контейнере нужно задать свойство display:flex
  • Justify-content - свойство выравнивания по главной оси
  • align-items - свойство выравнивания для поперечной оси
  • align-content - управления свойствами выравнивания по поперечной оси при переносе строк через flex-wrap
  • gap - значение промежутка между flex элементами
