Задание
Сделать страничку-таймер обратного отсчета. На страничке должно быть поле для ввода времени в секундах,
 кнопка "старт" и обратный отсчет должен начаться при нажатии на кнопку. Отсчет должен быть виден на 
страничке и обновляться каждую секунду. По окончании отсчета должен появиться тест
 "время вышло", или по желанию звуковое оповещение, таймер при этом должен остановиться.

 Объяснение кода:
HTML:

Содержит форму с полем для ввода времени в секундах, кнопку "Старт" и место для отображения таймера и сообщения о завершении.
CSS:

Простое оформление страницы: таймер и кнопки по центру, таймер в большом шрифте, сообщение выводится красным цветом.
JavaScript:

Использует асинхронность с помощью async/await и функцию delay(), которая задерживает выполнение на 1 секунду.
Функция startTimer() запускает обратный отсчет. Каждую секунду она обновляет значение таймера и отображает оставшееся время. Когда таймер достигает нуля, выводится сообщение "Время вышло!".
Для предотвращения неправильного ввода таймер обрабатывает некорректные значения и выводит сообщение об ошибке.
Как это работает:
Пользователь вводит время в поле ввода и нажимает "Старт".
Таймер начинает обратный отсчет, обновляя значение каждую секунду.
По истечении времени выводится сообщение "Время вышло!".
Этот код разделен по файлам и использует асинхронность в JavaScript для работы с таймером.
