в 17.1  подробное описание решения этой задачи
При загрузке страницы выполняется запрос к API.
Первые 15 задач из ответа добавляются в To-Do список.
Задачи автоматически добавляются как невыполненные.

Что происходит:
index.html: содержит основные элементы для To-Do листа, включая input для добавления новых задач, кнопки для фильтрации задач, и список задач.
styles.css: стилизует элементы страницы. Стили включают внешний вид кнопок, списка задач и состояния завершенных задач.
script.js:
Загружает задачи через fetch из API и добавляет их в список невыполненных дел.
Поддерживает возможность добавления пользовательских задач и фильтрации (все, выполненные, невыполненные).
Задачи могут быть отмечены как выполненные по нажатию на них.
Теперь страница To-Do автоматически загрузит 15 задач из API и отобразит их вместе с функциональностью добавления и фильтрации.
