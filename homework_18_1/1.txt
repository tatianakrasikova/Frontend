задание
Поработайте с двумя новыми APi:

одна дает случайные факты о кошках https://catfact.ninja/fact другая случайные фото кошек https://api.thecatapi.com/v1/images/search?size=med&mime_types=jpg&format=json&has_breeds=true&order=RANDOM&page=0&limit=1

заберите данные из API и отобразите на странице используя синтаксис async / await.

добавьте кнопку для обновление факта и кошки


Объяснение:
HTML (index.html):

Здесь мы создаем контейнер с заголовком, двумя блоками: один для факта, другой для изображения, и кнопкой для обновления контента.
Блоки факта и изображения имеют свои ID (#cat-fact и #cat-image), чтобы мы могли к ним обращаться и обновлять через JavaScript.
CSS (styles.css):

Стилизация добавлена для того, чтобы сделать страницу более приятной и аккуратной. Включены стили для выравнивания, кнопки, контейнеров и изображений.
JavaScript (script.js):

Мы используем async/await для асинхронных запросов к API.
getCatFact запрашивает случайный факт о кошках с API catfact.ninja.
getCatImage запрашивает случайное изображение кошки с API thecatapi.com.
Функция updateCatContent вызывает обе функции для получения факта и изображения, а затем обновляет HTML на странице.
Мы также добавили обработчик события для кнопки "Refresh", чтобы при каждом клике загружались новые данные.
Теперь у вас есть страничка, которая отображает случайный факт о кошках и случайное изображение кошки. Вы можете обновлять содержимое по нажатию на кнопку "Обновить".
